NAME

SSL_SESSION_get0_peer - get details about peer's certificate for a session

SYNOPSIS

 #include <openssl/ssl.h>

 X509 *SSL_SESSION_get0_peer(SSL_SESSION *s);

DESCRIPTION

SSL_SESSION_get0_peer() returns the peer certificate associated with the session s or NULL if no peer certificate is available. The caller should not free the returned value (unless RETURN VALUES

SSL_SESSION_get0_peer() returns a pointer to the peer certificate or NULL if no peer certificate is available.
