NAME

DH_new, DH_free - allocate and free DH objects

SYNOPSIS

 #include <openssl/dh.h>

 DH* DH_new(void);

 void DH_free(DH *dh);

DESCRIPTION

DH_new() allocates and initializes a DH structure.

DH_free() frees the DH structure and its components. The values are erased before the memory is returned to the system. If dh is NULL nothing is done.

RETURN VALUES

If the allocation fails, DH_new() returns NULL and sets an error code that can be obtained by SEE ALSO

ERR_get_error(3), DH_generate_key(3)
